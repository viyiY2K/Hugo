## 3.2.1 存储器（storage）

1.6 小节设计的 RAM 是一种易失性存储器(volatile memory)，一般来说电脑的内存是非永久性的（non-permanent），断电后数据会丢失。

存储器是一种非易失性（non-volatile）的存储介质，断电不会丢失，直到数据被删除或覆盖。

在以前“易失性”的速度快，“非易失性”的速度慢，但随着技术发展两者差异逐渐缩小。
